    把自己创建的包上传到npm中 ，也可以下载 ？
    1. 创建文件夹，命名规范，
    2. 生成package.json ： npm init -y
    3. 创建一个index.js入口文件
    1. 去npm官网注册账号： https://www.npmjs.com/
    2. 镜像地址改为npm的镜像地址： npm config set registry https://registry.npmjs.org/
    3. npm login ：输入npm账号密码邮箱
    4. 上传 ： npm publish
    下载包改为之前的淘宝镜像 ：npm config set rigistry https://registry.npm.taobao.org/

    怎么在包里面创建自己的命令？

    1.在package.json中加入 ‘bin’ 字段

    "bin":{
    //  my_name ：命令   my.js 运行命令的时候执行的js文件
    "my_name":"my.js"  
}

    1. 在js文件最上面添加一句话

      #!usr/bin/env node
指定运行node---js

    2. 推到线上，执行命令
